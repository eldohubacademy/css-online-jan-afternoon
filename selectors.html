CSS Selectors

CSS selectors provide a way to target an element or a group of elements for styling

Basic Selectors

  • Element/Type Selector -- using the element's tag name - styles all elements of that type in the page.
  • Class Selector - allows styling elements of same class(class attribute)
  • ID Selector - stlye one element(id are unique identifiers for an element)

Grouping selectors

Multiple selectors sharing the same declaration block

Combining Selectors

  • Descendant Selector(space) - all elements inside an element
  • Child selector(>) -targets only direct children
  • Adjacent sibling selector (+)
  • General sibling selector (~)
