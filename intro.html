Introduction to CSS

CSS- Cascading Style Sheets - a programming language used to enhance the look/presentation of a website hence building a nice looking user interface and therefore improving user experence.

CSS helps handle colors, positions and sizes of page content.

CSS Syntax - How do we write css


        
            selector{
                property: value;
                property: value;
                property: value;
            }
        
             

        
Example

        
            h1{
                color: green;
                text-decoration: underline;
            }

Where do we put our CSS

External CSS - put your code in .css file then link to the file from your html

Internal CSS - We put the styles in a style element

Inline CSS -- for styling individual elements using the style attribute
