CSS Units and Measurements

In CSS we can provide lengths and sizes using different units.

  • Absolute/Fixed units

    px, cm , in

  • Relative unit

    em, rem, %, vh, vw

    em - a unit relative to the font-size of the parent element

    rem - a value related to the font-size of the root element(html)

    vh - viewport height - availabe screen -- 100vh => full height

    vw - viewport width - availabe screen -- 50vw => half width of the screen

    % - percentage - in relation to the width/height of the parent element
